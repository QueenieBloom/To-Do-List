Todo-list Versão 1

O projeto consiste em uma aplicação web de lista de tarefas (To-do list), desenvolvida em HTML, CSS e JavaScript. A página web possui um formulário para adicionar novas tarefas, exibindo uma mensagem de alerta caso o limite de 15 tarefas seja atingido. Cada tarefa é representada por uma caixa de seleção, uma descrição e um botão de exclusão. As tarefas são armazenadas localmente no navegador utilizando o localStorage, permitindo que persistam mesmo após o recarregamento da página. Além disso, há uma funcionalidade que impede a adição de novas tarefas quando o limite é alcançado e remove a tarefa tanto visualmente quanto do armazenamento local quando o botão de exclusão é acionado.

Todo-list Versão 2

O projeto é uma aplicação web de lista de tarefas desenvolvida em HTML, CSS e JavaScript. A página apresenta duas seções principais: uma para criar novas tarefas e outra para exibir a lista de tarefas existentes. O formulário de criação de tarefas permite aos usuários adicionar uma descrição e escolher uma categoria entre "Personal" e "Business". As tarefas são exibidas em um layout organizado, onde cada tarefa possui uma caixa de seleção para marcar como concluída, uma bolha colorida indicando a categoria, a descrição da tarefa e botões para editar e excluir a tarefa. As informações das tarefas são armazenadas localmente no navegador usando o localStorage, garantindo a persistência dos dados mesmo após o recarregamento da página. A aplicação permite também editar o conteúdo das tarefas e excluí-las, mantendo a interface sempre atualizada conforme as interações do usuário.
